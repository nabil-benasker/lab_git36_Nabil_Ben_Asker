Outils 1 : Git

Les commandes de base

  • git init : initier le repot git en local
  • git status : lister les chargements dans le repertoire de travail
  • git add --all : ajouter les modifications à la staging area
  • git commit -m"Premier commit" : sauvegarde de cette version du projet
  • git log : permet de consulter les differents commits(etendue)
  • git log --oneline : permet de consulter les commits (résumé)
  • git branch dev : créer une nouvelle branch dev
  • git branch : consulter toutes les branches
  • git checkout dev : basculer vers la branche dev
